# Ejemplos Visuales de Temas

Esta guia te muestra ejemplos de como se ven los colores de cada tema y como usarlos en tu codigo.

## Como se usan los colores en el codigo

En este proyecto usamos Tailwind CSS, que nos permite usar los colores del tema con clases especiales:

```jsx
// Ejemplo de un boton con colores del tema
<button className="bg-accent text-white hover:bg-accent-dark">
  Haz clic aqui
</button>

// Ejemplo de una tarjeta con fondo
<div className="bg-primary-dark text-foreground">
  <h2 className="text-accent">Titulo</h2>
  <p className="text-foreground-light">Descripcion</p>
</div>
```

## Consejos para Combinar Colores

### Regla 60-30-10

Una buena regla de diseno es la regla 60-30-10:

- **60%** - Color dominante (primary)
- **30%** - Color secundario (primary-dark)
- **10%** - Color de acento (accent)

### Contraste para Accesibilidad

Siempre verifica que haya suficiente contraste entre texto y fondo:

- **Texto normal**: Minimo 4.5:1
- **Texto grande** (18px+): Minimo 3:1
- **Elementos interactivos**: Minimo 3:1

Usa [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) para verificar.

### Estados Interactivos

Para elementos interactivos (botones, enlaces), usa las variaciones de color:

```jsx
// Estado normal
className="bg-accent"

// Estado hover
className="hover:bg-accent-dark"

// Estado activo/seleccionado
className="bg-accent-light"

// Estado deshabilitado
className="bg-muted cursor-not-allowed"
```

## Personalizacion Avanzada

Si quieres ajustar un color especifico sin crear un tema completo:

1. Abre el archivo del tema activo (ej: `src/styles/themes/classic.js`)
2. Modifica solo el color que necesites
3. Guarda y recarga el navegador

Ejemplo:

```javascript
// Cambiar solo el color de acento
export const classicTheme = {
  primary: '#FFFFFF',
  'primary-dark': '#F5F5F5',
  accent: '#ff6b6b',  // Cambiado a rojo coral
  // ... resto de colores igual
};
```

Recuerda: Si modificas un tema existente, afectara a todos los que usen ese tema. Si quieres experimentar, mejor crea una copia del tema con otro nombre.
